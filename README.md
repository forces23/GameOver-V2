# GameOverVault

## Backend

Running the FastAPI backend is as easy as navigating to the backend directory in the terminal and running these commands

```Bash
source .venv/bin/activate

uvicorn main:app --reload
```

once you run this you should be able to access the API end points by using the url

```bash
http://127.0.0.1:8000/
```

To access the FastAPI Interactive API Docs (provided by Swagger) go to this url

```bash
http://127.0.0.1:8000/docs#/
```

Or go here to access the Alt Auto Docs (provided by Redocs)

```bash
http://127.0.0.1:8000/redoc
```
